# Mobius Schema

Canonical shared contracts for the Mobius Organizational Intelligence Operating System.

## Mission

Define once. Reuse everywhere.

Mobius Schema is the shared language of the platform. Core services should import platform contracts from this repository instead of redefining models independently.

## Owns

- Organizations
- Identities
- Departments
- Repository DNA
- Platform Change Proposals
- Memory Events
- Pulse Metrics
- Event Bus messages
- Task envelopes
- Audit records
- Synchronization reports

## Rule

Every shared concept has exactly one canonical definition.
